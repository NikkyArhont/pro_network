# Product Requirements Document (PRD)

## О продукте
Мобильная социальная сеть для профессионалов. Платформа объединяет возможности личного общения и построения деловых связей, позволяя пользователям разделять свои контакты по категориям, вести профессиональную активность, обмениваться опытом и находить новые полезные знакомства в едином и удобном пространстве.

## Цель продукта
Предоставить пользователям инструмент для бесшовного объединения личного и профессионального нетворкинга без размытия границ приватности. Дать возможность комфортно общаться с семьей, управлять профессиональной репутацией (через рекомендации) и следить за профильным контентом в одном приложении.

## Ключевые пользовательские сценарии (User Flows)
1. **Управление идентичностью:** Регистрация, заполнение базового личного профиля и отдельная настройка профессионального профиля (с навыками и опытом) для деловых контактов.
2. **Категоризация контактов:** Сохранение новых и существующих контактов с жестким распределением по спискам (семья, друзья, полезные связи), что определяет их доступ к вашей информации.
3. **Обмен контентом (с учетом приватности):** Создание поста с выбором аудитории (например, статья только для категории "Полезные связи" или фото только для "Семья").
4. **Профессиональный поиск и найм:** Поиск специалиста или компании по ключевым навыкам, изучение оставленных ему рекомендаций, начало диалога.
5. **Построение репутации:** Запрос или написание текстово-рейтинговых рекомендаций коллегам, с которыми пользователь успешно завершил проекты.

## Основные разделы приложения
1. **Профиль** (Личная информация, профессиональное портфолио, управление конфиденциальностью).
2. **Контакты** (Списки по категориям, поиск пользователей, входящие запросы).
3. **Лента** (Посты, истории, фильтрация контента по категориям контактов).
4. **Мессенджер** (Личные и групповые чаты с контактами).
5. **Компании** (Бизнес-профили, поиск работы / предоставление услуг, подписки).
